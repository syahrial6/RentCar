# Artha Royal Group - Aplikasi Sewa Mobil Pontianak

## 📌 Deskripsi Proyek

Aplikasi **Artha Royal Group** adalah website landing page untuk layanan penyewaan mobil di wilayah **Pontianak**. Dibangun menggunakan **Next.js**, website ini berfungsi sebagai media informasi dan katalog kendaraan, serta mengarahkan pelanggan untuk melakukan pemesanan melalui **WhatsApp**.

## 🚀 Fitur Utama

- 🏠 Halaman landing page sederhana dan informatif
- 🚗 Katalog mobil yang tersedia untuk disewa
- ℹ️ Section tentang perusahaan (About)
- 📞 Booking langsung melalui WhatsApp
- 📱 Desain responsive (mobile-friendly)

## 🛠️ Teknologi yang Digunakan

- **Framework**: Next.js
- **Frontend**: React.js
- **Styling**: Tailwind CSS / CSS Modules

> Catatan: Proyek ini tidak menggunakan backend, database, maupun sistem autentikasi.

## ⚙️ Instalasi & Menjalankan Proyek

### 1. Clone Repository

```bash
git https://github.com/syahrial6/RentCar.git
cd RentCar
```

### 2. Install Dependencies

```bash
npm install
# atau
yarn install
```

### 3. Jalankan Development Server

```bash
npm run dev
```

Buka di browser: [http://localhost:3000](http://localhost:3000)

## 📦 Build untuk Production

```bash
npm run build
npm start
```
